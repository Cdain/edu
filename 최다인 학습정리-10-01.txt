4. 변수란 무엇인가?
변수(Variable)는 값을 저장할 수 있는 메모리의 특정 번지에 붙이는 이름이다.

5. 변수선언의 의미는?
변수에 어떤 타입의 데이터를 저장할 것인지, 변수 이름이 무엇인지를 선언하는 것이다.

6. Java의 자료형 작성 (8형제) 에 대하여 설명하시오.
boolean, char, byte, short, int, long, float, double
boolean 은 참과 거짓을 나타내는 자료형이며 1바이트의 크기를 가졌고 -128~127까지 표현된다.
char는 문자를 나타내는 자료형이며 2바이트의 크기를 가졌다.
byte, short, int, long 은 정수를 나타내는 자료형이며, 각각 1바이트, 2바이트, 4바이트, 8바이트의
크기를 가졌고 그 중 int 는 -2,147,483,648~2,147,483,647까지 표현된다.
float, double은 실수를 나타내는 자료형이다.

7. int 형 범위는?
-2,147,483,648~2,147,483,647

8. int num; 을 메모리로 설명해보자.

메모장에 그림 그리는 방법을 모르겠어요.

9. java 언어를 창시한 사람은?
제임스 고슬링

10. JDK란 무엇이며, 어디서 다운로드 받으며, OS별로 버전이 있는 까닭은?
Java Development Kit의 약자이며, 자바 개발 도구라고 한다. 자바 소프트웨어 개발 환경인데 응용
프로그램 개발을 위한 도구들이 포함된다. OS별로 버전이 있는 까닭은 OS가 다르면 JVM이 다르기 때문이다.

11. JVM 무엇의 약자이며,이란 무엇인가?
자바가상머신(Java Virtual Machine)의 약자이다. 자바로 만들어진 프로그램을 실행할때
메모리에 주소를 정해준다..?

12.아래에 대하여 설명하시오.
write once, run everywhere.
JAVA 언어의 슬로건(표어)이며 줄여서 WORE 한번 작성된 코드로 여러 플랫폼에서 실행할 수 있다는 뜻.
